# 互動式地圖資訊

## 檔案位置
互動式地圖檔案位於 `deliverables/interactive_map.zip`

## 檔案大小
- 原始 HTML 檔案：159 MB
- 壓縮後 ZIP 檔案：50.5 MB
- GitHub 限制：100 MB ✅

## 使用方法

### 方法1：下載並解壓縮
1. 在 GitHub 下載 `deliverables/interactive_map.zip`
2. 解壓縮 ZIP 檔案
3. 雙擊解壓後的 HTML 檔案
4. 使用瀏覽器查看互動式地圖

### 方法2：直接下載整個專案
1. 下載整個專案：`git clone https://github.com/chengzong1023/hw3.git`
2. 解壓縮 `deliverables/interactive_map.zip`
3. 開啟 HTML 檔案

## 地圖功能
- 三級緩衝區顯示（高/中/低風險）
- 河川面顯示（藍色）
- 避難所點位（依風險等級著色）
- 鄉鎮區界線
- 點擊避難所顯示詳細資訊
- 圖層控制功能

## 技術規格
- 使用 Folium 製作
- 包含 5,889 個避難所點位
- 使用真實收容人數資料
- 支援互動式操作
