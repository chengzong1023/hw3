# 互動式地圖資訊

## 檔案位置
互動式地圖檔案位於 `temp_files/risk_map_interactive.html`

## 檔案大小
- 檔案大小：約 159 MB
- 超過 GitHub 單檔案大小限制 (100 MB)

## 解決方案
由於檔案太大無法直接上傳到 GitHub，請使用以下方式：

### 方法1：本地執行
1. 下載整個專案
2. 在本地開啟 `temp_files/risk_map_interactive.html`
3. 使用瀏覽器查看互動式地圖

### 方法2：使用 GitHub Pages
如果需要線上展示，可以：
1. 啟用 GitHub Pages
2. 上傳 HTML 檔案到 Pages

### 方法3：縮小地圖檔案
可以考慮：
- 減少地圖的詳細程度
- 使用較少的地圖圖層
- 優化 HTML 檔案大小

## 地圖功能
- 三級緩衝區顯示（高/中/低風險）
- 河川面顯示（藍色）
- 避難所點位（依風險等級著色）
- 鄉鎮區界線
- 點擊避難所顯示詳細資訊
- 圖層控制功能

## 技術規格
- 使用 Folium 製作
- 包含 5,889 個避難所點位
- 使用真實收容人數資料
- 支援互動式操作
